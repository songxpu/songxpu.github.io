信息收集

netdiscover -r 192.168.64.0/24 => 192.168.64.148

nmap -oN nmap.txt -A 192.168.64.148

dirb 192.168.64.148 => /vendor/ /wordpress/

其中wordpress无法正常打开，vendor存在目录遍历，并发现phpmailer 5.2.16，存在漏洞
