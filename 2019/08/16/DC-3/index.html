拓展

渗透过程中了解到或复习到的小知识点，记录下面便于今后查阅

Ubuntu开启ssh

  • 安装openssh
    sudo apt-get install openssh-server
  • 查看ssh状态
    ps -aux|grep ssh
  • 启动服务
    service ssh start

Ubuntu用户管理(普通用户->root)

  • 创建用户 useradd test

  • 设置密码 passwd test

  • 修改/etc/passwd：root权限下修改uid：0

  • 用户加入sudo组：在/etc/sudoers文件里给该用户添加权限

John破解Linux密码教程

  • 破解Linux下passwd、shadow中的密码

因此，查看上一次爆破结果：john --show shadow
